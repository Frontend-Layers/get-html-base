# get-html-base

Get HTML Base by command line

## Features

- deploy [html-base](https://github.com/andreymatin/html-base) by command line

## Installation

```
npx get-html-base project-name
```

## How to use

1. Go to the project folder

2. Install dependencies

```
npm i
```
